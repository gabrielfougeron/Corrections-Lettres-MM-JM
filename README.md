# Correction lettres MM-JM
Repo public pour la correction collaborative des lettres de fiançailles des Marie-Madeleine et Jean-Marie. Les fichiers nécéssaires à la construction des documents sont [à cette addresse](https://github.com/gabrielfougeron/Lettres_Fiances_BM_BP) (repo privé).

# Guides des versions exportées

L'ensemble des versions exportées est le produit cartésien des paramètres suivant :

- Le format de la page. Il y a pour l'instant 3 formats (A4, A5, Kindle). Le format privilégié est le format A5.
- Avec ou sans images optionnelles. Les versions avec absolument toutes les images sont estampillées `-full`, et les versions avec uniquement les images les plus importantes sont estampillées `-textonly`.
- Images en couleurs ou en noir et blanc. Les versions en noir et blanc sont environ 20x plus légères, et sont donc estampillées `-lightweight`.
- Les versions où chaque page du document correspond à une page de lettre, et celles où les pages transcrites ont été fusionnées, et où les images sont reléguées en fin de lettre. Ces dernières sont estampillées `-fusepages`
